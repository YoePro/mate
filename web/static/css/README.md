# CSS structure

This folder contains CSS for the MATE frontend.

## Files

- `reset.css` normalizes browser defaults.
- `main.css` contains variables, typography, and base styling.
- `layout.css` contains page layout, workspace layout, and panels.
- `components.css` contains reusable UI components.
- `graph.css` contains node and link styling.
- `toolbox.css` contains toolbox styling.
